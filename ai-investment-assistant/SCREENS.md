```

│ KabuAI        │  ダッシュボード              │
│               │                             │
│ ◉ ダッシュ    │  ┌────────┐ ┌────────┐     │
│   ボード      │  │総評価額 │ │評価損益 │     │

│   リオ        │  └────────┘ └────────┘     │
│ ○ AI チャット │                             │
│               │  ┌──────────────────────┐  │
│               │  │ アセット配分 円グラフ  │  │
│               │  │  日本株 60%           │  │
│               │  │  米国株 30%           │  │
│               │  │  ETF   10%           │  │
│               │  └──────────────────────┘  │
│               │                             │
│               │  保有銘柄一覧               │
│               │  ┌──────────────────────┐  │
│               │  │ 銘柄 │ 数量 │ 損益   │  │
│               │  │ 7203 │ 100  │+5,000  │  │
│               │  │ AAPL │  10  │-2,000  │  │
│               │  └──────────────────────┘  │
└───────────────┴─────────────────────────────┘
```

**表示データ:**
| 項目 | 算出方法 |
|------|---------|
| 総評価額 | Σ (現在株価 × 保有数) × 為替レート |
| 評価損益 | 総評価額 − 総取得金額 |
| 損益率 | 評価損益 ÷ 総取得金額 × 100 |
| アセット配分 | 銘柄の市場（国内/米国）別比率 |

**コンポーネント:** `PnLSummary`, `AllocationChart`, `HoldingTable`

```
│               │  └─────────────────────────┘       │
└───────────────┴───────────────────────────────────────┘
```

**クイック質問ボタン（定型文）:**
```
[ポートフォリオ全体の評価]
[含み損銘柄の対処法]
[分散状況を確認]
[今月の相場動向について]
```

**システム動作:**
1. ページ初期表示 → ポートフォリオ情報を取得しシステムプロンプトに埋め込み
2. メッセージ送信 → `/api/chat` へ POST（ストリーミング応答）
3. AI 応答を逐次表示（SSE / ReadableStream）
4. チャット履歴を `chat_messages` テーブルに保存

**AI ガードレール:**
- 根拠のない推奨 → 「数値的根拠が不足しているため推奨はできません」
- 全財産を一点集中など危険な提案 → 「リスク分散の観点からお勧めできません」
- 毎回の応答末尾に免責文を付加

## 共通コンポーネント

### サイドバー
```
KabuAI ロゴ
─────────
📊 ダッシュボード
💼 ポートフォリオ
💬 AI チャット
─────────
[ユーザーアバター]
settings / ログアウト
```

### ローディング状態
- 株価取得中: Skeleton UI（シマー）
- AI 応答中: 「...」アニメーション + ストリーミングテキスト

### エラー状態
- 株価取得失敗: 「株価データを取得できませんでした。前回の取得値を表示中」
- AI エラー: 「AIに接続できませんでした。再試行してください」
